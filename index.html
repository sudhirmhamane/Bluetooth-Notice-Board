About the Project

This project allows users to control a digital notice board remotely using a smartphone. It replaces traditional notice boards with a more efficient and environmentally friendly solution using Arduino and Bluetooth technology.

Project Details

This system uses an Arduino microcontroller, Bluetooth module, and an LCD display to create a digital notice board. The system can be controlled through a smartphone app, which sends messages to be displayed on the board.

  • Technology Used: Arduino, Bluetooth, LCD Display
  • Functionality: Remote message updating, customizable display settings
  • Application: Educational institutions, offices, public places

Project Components

Arduino MCU

The Arduino board acts as the main controller for the project, allowing it to receive data from the smartphone and control the display

Bluetooth Module

This module facilitates wireless communication between the smartphone and the Arduino.

Display Screen

The display screen is used to show messages by giving the input from the smartphone.

Breadboard & Wires

A breadboard is used for prototyping without soldering, while jumper wires connect various components.

Resistors

Passive components used to control voltage and current in the circuit.
